
This file contains Ezoic setup instructions for the site administrator:

1. Replace 'your-ezoic-site-id' in index.html with your actual Ezoic Site ID
2. Replace 'your-ezoic-verification-code' in index.html with your Ezoic verification code
3. The following ad slots are configured in the site:
   - top-banner (horizontal banner at top)
   - sidebar-ad (rectangle sidebar ad)
   - mid-content (horizontal mid-content ad)
   - blog-header (blog page header ad)
   - blog-sidebar (blog page sidebar ad)
   - blog-mid-content (blog mid-content ad)
   - bottom-content (bottom content ad)

4. In your Ezoic dashboard, create ad placements that match these slot names
5. Ezoic will automatically optimize ad placements and formats
